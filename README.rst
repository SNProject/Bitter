Bitter Documentation
====================
Bitter is a simple but powerful analytics library

Installation
------------
Use `Composer <https://github.com/composer/composer/>`_ to install : `free-agent/bitter`.

Bitter uses `Redis <http://redis.io>`_ (version >=2.6).
